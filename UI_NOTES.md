# UI Notes

## Most Important Thing
The most important thing is showing the total cycle price clearly and instantly.

## Fast Repetitive Use
Since the salesperson uses it many times daily, dropdown selection and instant pricing make the process faster.

## Invalid Combination Handling
If an invalid combination is selected (example: tubeless tyre without compatible rim), the UI should show an error message and prevent submission.

## Improvement with More Time
With more time, I would build a fully connected web UI with backend integration and real-time validation.
